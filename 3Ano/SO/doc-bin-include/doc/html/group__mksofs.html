computes the structural division of the disk

The following should be observed:

  • if itotal is zero, the value ntotal / 16 should be used as a start value, where / stands for the integer division;
  • itotal must be always lower than or equal to ntotal / 8;
  • itotal must be rounded up to be multiple of IPB;
    Parameters
    [in]ntotalTotal number of blocks of the disk
    [in,out]itotalTotal number of inodes
    [out]nbrefNumber of reference data blocks in a newly-formatted disk

Fill in the fields of the superblock.

The following should be observed:

  • the magic number should be put at 0xFFFF;
  • The tail cache should be left empty;
  • the head cache should be left filled (totally, if possible).
    Parameters
    [in]namevolume name
    [in]ntotalthe total number of blocks in the device
    [in]itotalthe total number of inodes
    [in]nbrefNumber of reference data blocks in a newly-formatted disk

◆ soFillInodeTable()

void sofs19::soFillInodeTable ( uint32_t  itotal,
bool  set_date = true 
)

Fill in the blocks of the inode table.

The following should be observed:

  • inode table starts on block number 1;
  • inode 0 must be filled assuming it is used by the root directory;
  • all the other inodes are free;
  • the first free inode should be inode number 1, the second inode number 2, and do forth;
  • the last free inode should point to NullReference.
    Parameters
    [in]itotalthe total number of inodes

Fill in the data blocks containing references to free data blocks in a newly-formatted disk.

The following should be observed:

  • the list of free data blocks is initially sorted in ascending order;
  • the first references of the list are in the head cache;
  • non-used cells must be filled with pattern NullReference;
  • there is a possibility that for function to do nothing, in case all references fit in the head cache.
    Parameters
    [in]ntotalthe total number of blocks in the device
    [in]itotalthe total number of inodes
    [in]nbrefNumber of reference data blocks in a newly-formatted disk

◆ soFillRootDir()

void sofs19::soFillRootDir ( uint32_t  itotal)

Fill in the root directory.

The following should be observed:

  • in a newly-formatted disk the root directory occupies a single, the one immediately after the inode table;
  • the first two slots must be filled in with "." and ".." directory entries.
  • the other slots must be cleaned: field name filled with zeros and field inode filled with NullReference.
    Parameters
    [in]itotalthe total number of inodes
